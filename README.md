# exercicio_lista_itinerario

## Descrição
Este projeto tem como objetivo implementar uma lista de itinerários, permitindo o gerenciamento de rotas e destinos de forma organizada.  
O código foi estruturado seguindo boas práticas de organização em diretórios e documentação.

## Requisitos
- **Compilador:** GCC versão 11 ou superior  
- **IDE recomendada:** Visual Studio Code  
- **Extensões úteis:** C/C++ (Microsoft), CMake Tools (opcional)
